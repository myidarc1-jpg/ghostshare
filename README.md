## How to Use
- Install Node.js (LTS) and npm.
- Run `npm install` to install dependencies.
- (Optional) Copy `.env.example` to `.env` and adjust configuration.
- Start the app locally with `npm run start` (or `npm run dev` for development).
- Access the Creator Dashboard at the provided local URL (usually http://localhost:3000).
